1. Завдання на знання темізації

Встановити Підтему на базі теми ZEN https://www.drupal.org/project/zen . Написати препроцес для page і передати в page.tpl.php змінну, 
яка буде виводити теперішню дату в форматі d.m.Y

2. Завдання на знання API

Створити модуль, створити сторінку через hook_menu. На сторінці створити кастомну форму з полями Імя*, Емейл*, Коментар. 
При сабміті форми вся інформація відсилається на емейл. Для форми написати валідацію, яка буде перевіряти правильність введення емейлу.

3. Завдання на знання головних контрібних модулів Друпала

Створити тип контенту product. В ньому створити поля ціна, фото, код, опис, к-ть. Створти через модуль Devel 500 тестових продуктів. 
Створити вювку у вигляді таблиці, яка буде відображати список всіх продуктів, сортувати по ціні і по кількості товари, фільтрувати по коду через ajax. 
У вювці повинні відображатися сторінки по 50 товарів.
